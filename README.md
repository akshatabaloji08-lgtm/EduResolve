# EduResolve 🎓
### Smart Campus Issue Resolution Portal

A web-based complaint management system where students can 
register, login, and submit campus issues. Admin can track, 
manage, and resolve complaints in real-time.

## Features
- Student Registration & Login
- OTP-Based Password Reset
- Complaint Submission with Category & Description
- Real-Time Complaint Status Tracking
- Admin Dashboard for Complaint Management
- Email Notifications via PHPMailer
- PDF Report Generation via FPDF

## Tech Stack
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL
- Libraries: PHPMailer, FPDF
